How do you select an element from the DOM ?

Use an ID to Select an Element with . getElementById() One way of selecting an element is by using the . getElementById() method and passing the desired ID as an argument.

What is event delegation in the context of the DOM, and why is it useful ?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated
